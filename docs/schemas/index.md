# Schemas

The Story as Code specification defines the following YAML schemas (v0.1.0):

| Schema | Description |
|--------|-------------|
| [World](world.md) | Root `world.yaml` metadata |
| [Node](node.md) | Nodes: characters, locations, objects, events, concepts |
| [Edge](edge.md) | Edges: relationships between nodes |
| [Frame](frame.md) | Temporal frames and time topologies |
| [Lens](lens.md) | Narrative perspective and filters |
| [Format](format.md) | Output format definitions |
| [Arc](arc.md) | Story arcs with milestones |
| [Constraint](constraint.md) | World rules and validation |
| [Schema](schema.md) | User/community schema definitions |
| [Core Types](core-types.md) | Built-in node and edge types |
| [Derivation Meta](derivation-meta.md) | Derivation validation contracts |
| [Variant Meta](variant-meta.md) | Parallel world version metadata |
